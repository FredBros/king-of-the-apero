# King of the Apero (Rois du Ring)

Digital adaptation of the board game "Les Rois du Ring" (Chien Sauvage).

## Project Overview

*   **Genre:** Turn-based strategy, 1v1, fast-paced.
*   **Gameplay:** Grid movement (5x5 or 6x6) using a card system (Move, Attack, Throw).
*   **Target Platforms:** Mobile and Web (PWA).
*   **Current Status:** Phase 1 - Mechanical Prototype (POC).

## Tech Stack

*   **Engine:** Godot Engine 4.x
*   **Language:** GDScript
*   **Art Style:** 3D Low-Poly
